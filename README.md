# Contract Intelligence Platform 
 
AI-powered legal analysis for Indian SMEs - Analyze contracts in seconds, identify risks, and get actionable insights. 
 
## Features 
 
- Risk Scoring: Automated 0-100 risk assessment 
- Entity Extraction: Identifies parties, amounts, dates 
- Clause Classification: Categorizes obligations, rights, prohibitions 
- Compliance Checking: Indian Contract Act validation 
- Export: Download analysis as JSON 
 
## Tech Stack 
 
- Streamlit, Python, Plotly 
- NLP: Regex-based pattern matching 
- Document parsing: PyPDF2, pdfplumber 
 
## Installation 
 
pip install -r requirements.txt 
streamlit run app.py
